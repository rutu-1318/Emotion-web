About EmotionAI

EmotionAI is an intelligent multi-modal system that detects human emotions using Text, Audio, and Facial expressions. It leverages state-of-the-art machine learning models to make accurate predictions in real time or from uploaded data.

üìù Text Emotion Detection

This module analyzes user input or uploaded `.txt` files and classifies emotions based on semantic and contextual features.

  • Model: Logistic Regression
  • Features: TF-IDF vectorization
  • Emotions: Happy, Sad, Angry, Neutral, Fear

üéß Audio Emotion Detection

Extracts features from speech like MFCC, pitch, and energy, and uses them to determine emotional states.

  • Model: Random Forest
  • Features: MFCCs, Chroma, Zero-Crossing Rate
  • Emotions: Happy, Sad, Angry, Fear, Surprise

üì∑ Face Emotion Detection

Detects emotions from facial expressions using deep learning and computer vision techniques.

  • Model: CNN (Convolutional Neural Network)
  • Features: Real-time face landmarks + expression features
  • Emotions: Happy, Sad, Angry, Disgust, Surprise, Neutral
